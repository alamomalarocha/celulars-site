# CDVS External Reference Report

Pesquisa complementar para o CELULARS Device Visual System.

Data da pesquisa: 2026-06-29

Escopo: consultar fontes externas indicadas pelo projeto para complementar a base visual e histórica dos modelos de iPhone, especialmente modelos antigos e cores ainda sem imagem oficial Apple/CDVS integrada.

## Resumo executivo

- O catálogo CELULARS atual contém 52 modelos, do iPhone original de 2007 ao iPhone 17e.
- As fontes externas consultadas ajudam principalmente como referência de cronologia, formato físico, câmera, frente/traseira e cores.
- Nenhuma imagem dessas fontes foi salva no projeto nesta etapa, porque não foi encontrada uma licença clara e segura para uso comercial direto no site.
- Getty Images deve permanecer apenas como fonte visual licenciável/referencial; não deve ser baixada sem compra/licença explícita.
- Reddit, Olhar Digital, Oficina da Net e Canaltech devem ser tratados como referência editorial/visual, não como fonte final de imagem comercial.
- A recomendação técnica é continuar usando imagens oficiais Apple quando disponíveis e, para modelos antigos/cores faltantes, produzir ilustrações próprias no padrão CDVS.

## Referências úteis por fonte

### Olhar Digital

Modelos encontrados:

- Do iPhone original ao iPhone 17, incluindo iPhone Air e iPhone 16e.
- A página organiza os modelos por ano, com seções específicas para modelos antigos como iPhone 3G, 3GS, 4, 4s, 5, 5c e 5s.

Cores encontradas:

- Não é uma fonte principal para nomes oficiais de cores.
- Útil para referência visual geral de acabamento em modelos antigos.

Imagens ou referências úteis:

- A página contém imagens de modelos históricos e modelos atuais.
- Útil para analisar proporção, câmera, bordas e diferenças visuais entre famílias.

Uso comercial:

- Não usar imagens no site sem autorização/licença.
- Recomendação: apenas referência.

### Oficina da Net

Modelos encontrados:

- Lista do iPhone original até modelos 2025, incluindo iPhone 17, iPhone Air, iPhone 16e, iPhone 16, iPhone 15 e linhas anteriores.

Cores encontradas:

- Algumas imagens indicam famílias completas, como iPhone 16 em todas as cores e iPhone 16 Pro em todas as cores.
- Útil para confirmar agrupamento visual, não como fonte oficial de cor.

Imagens ou referências úteis:

- Imagens de visão geral por geração.
- Boas referências para cards internos do CDVS: frente/traseira, câmera dupla, câmera tripla e diferenças entre Pro/base.

Uso comercial:

- Não usar imagens no site sem autorização/licença.
- Recomendação: apenas referência.

### Wikipédia

Modelos encontrados:

- Página de lista de modelos de iPhone, útil para conferência histórica e nomenclatura.

Cores encontradas:

- A página consultada não foi a melhor fonte para cores oficiais detalhadas.

Imagens ou referências úteis:

- A página de lista em português não apresentou imagens de aparelhos úteis para baixar; via API, os arquivos de imagem listados eram majoritariamente ícones/logos de interface da Wikipédia/Commons, não fotos de iPhone.

Uso comercial:

- Texto da Wikipédia pode ter licença reutilizável sob condições, mas imagens individuais no Commons precisam de verificação de licença caso a caso.
- Nenhuma imagem foi salva nesta etapa.
- Recomendação: referência histórica; não usar imagens sem auditoria individual.

## Recomendações para produção CDVS

1. Não importar imagens de Olhar Digital, Oficina da Net, Canaltech, Reddit ou Getty diretamente para o site.
2. Usar essas fontes apenas para guiar proporção, família visual, posição de câmeras, notch, Home Button e acabamento.
3. Continuar procurando imagens oficiais Apple em Apple Newsroom, Apple Support, Apple Store e páginas antigas arquivadas quando disponíveis.
4. Para iPhone 14 e anteriores, priorizar produção vetorial própria CDVS com base nas famílias visuais já documentadas.
5. Para Getty, só usar se uma imagem específica for licenciada formalmente e a licença permitir uso no site da CELULARS.
6. Para Wikimedia Commons, avaliar arquivo por arquivo antes de salvar: licença, atribuição, uso comercial, marcas registradas e qualidade visual.
