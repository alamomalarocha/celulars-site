# CELULARS Device Visual System (CDVS)

# Objetivo

Criar uma biblioteca própria da CELULARS para representar todos os iPhones utilizados no catálogo.

Não utilizar fotos misturadas.

Não depender de imagens oficiais.

Manter um padrão visual único.

---

# Padrão visual

Todos os aparelhos devem seguir exatamente a mesma composição.

Sempre:

* aparelho frontal
* aparelho traseiro
* mesma iluminação
* mesma sombra
* mesmo enquadramento
* mesma escala

---

# Famílias

Organizar por famílias:

* iPhone clássico
* Home Button
* Notch Dual
* Notch Triple
* Dynamic Island Dual
* Dynamic Island Triple
* Air
* SE

---

# Componentes

Documentar que cada aparelho possuirá componentes reutilizáveis:

* moldura
* tela
* vidro
* câmera
* flash
* LiDAR
* acabamento
* sombra

---

# Sistema de cores

As cores não serão desenhadas individualmente.

Cada aparelho utilizará uma paleta oficial.

Exemplos:

* Black Titanium
* Natural Titanium
* Desert Titanium
* Blue Titanium
* Pink
* Midnight
* Starlight
* etc.

---

# Wallpapers

Criar biblioteca separada.

Cada família terá wallpapers próprios.

Sem copiar Apple.

Inspirados apenas.

---

# Branding

Criar espaço reservado para assinatura CELULARS.

Discreta.

Sem exagero.

---

# Objetivo final

Toda a biblioteca deverá ser reutilizável em:

* Site CELULARS
* PHIXCELL
* Catálogos PDF
* Instagram
* Material gráfico
* Aplicativo futuro
