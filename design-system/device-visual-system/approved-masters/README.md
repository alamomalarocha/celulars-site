# Approved Masters — CDVS

Esta pasta guarda os Masters aprovados do CELULARS Device Visual System.

Cada Master representa uma referência visual oficial para uma família ou modelo de iPhone dentro do ecossistema CELULARS.

Os arquivos finais de cada Master serão organizados em sua própria pasta, com separação entre fonte, exportações, documentação e previews.

## Estrutura por Master

* `source/`: arquivos fonte do Master.
* `exports/`: arquivos finais exportados.
* `documentation/`: documentação técnica e visual.
* `previews/`: imagens de revisão e aprovação.
