# CDVS — CELULARS Device Visual System

O CELULARS Device Visual System (CDVS) é a biblioteca oficial de representação visual dos aparelhos usados pela CELULARS.

O objetivo é criar ilustrações próprias, consistentes e reutilizáveis para substituir fotos misturadas no catálogo e em materiais comerciais.

Cada aparelho deve seguir um padrão único de composição, escala, sombra, enquadramento e acabamento, mantendo aparência premium, limpa e minimalista.

O CDVS será usado no site CELULARS, PHIXCELL, catálogos PDF, Instagram, materiais gráficos e aplicações futuras.

## Estrutura

* `vectors/`: arquivos vetoriais fonte.
* `templates/`: modelos reutilizáveis por família.
* `colors/`: paletas oficiais dos aparelhos.
* `wallpapers/`: wallpapers próprios por família.
* `exports/`: versões finais exportadas.
* `previews/`: imagens de revisão visual.
* `references/`: referências visuais e técnicas.
* `documentation/`: regras e documentação do sistema.
