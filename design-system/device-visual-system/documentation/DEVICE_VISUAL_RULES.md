# DEVICE VISUAL RULES

Regras oficiais iniciais do CELULARS Device Visual System (CDVS).

## Composição

* Todos os aparelhos devem ter frente + traseira lado a lado.
* Fundo transparente.
* Mesma escala, mesma sombra, mesmo enquadramento.
* Visual premium, minimalista, limpo e sem aparência de IA.

## Fidelidade por Família

* Câmeras, notch, Dynamic Island, flash e LiDAR devem respeitar cada família de iPhone.
* A cor do aparelho deve mudar conforme a cor selecionada no catálogo.

## Branding

* A marca CELULARS deve aparecer apenas como assinatura discreta, nunca como elemento chamativo.
* Não usar textos como Novo, CPO ou preço dentro das imagens.

## Reutilização

As imagens devem poder ser reutilizadas em:

* site CELULARS
* PHIXCELL
* PDFs
* Instagram
* materiais comerciais

## Formatos Finais

Formatos finais recomendados:

* SVG fonte
* PNG transparente
* WEBP otimizado
